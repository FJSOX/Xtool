## 功能特性

### 编码转换工具
- 支持多文件选择
- 支持自动检测文件编码
- 支持自定义编码格式
- 批量转换功能

### 文件重命名工具
- 支持多文件选择
- 支持多种后缀格式
- 支持自定义后缀
- 批量重命名功能

## 开发环境
- Python 3.8+
- PyQt5
- chardet

## 安装依赖
```bash
pip install PyQt5 chardet
```

## 使用说明
1. 运行主程序：
```bash
python 小工具合集.py
```

2. 在左侧菜单选择所需工具
3. 根据界面提示操作

## 项目结构
